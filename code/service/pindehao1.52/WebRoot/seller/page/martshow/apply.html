󰂶

为规范商家运营行为，提升运营质量，即日起，创建品牌特卖专场将进行包括但不仅限于以下限制：

1. 连续两个专场销售额均低于10000元的品牌不可创建专场；

2. 品牌综合评分低于4.2分的品牌不可创建专场；

3. 超过90天未运营的品牌不可创建专场；

4. 品控质检超过2次不合格的品牌不可创建专场；
